# TravelVista — 旅游目的地浏览网站 PRD

## 1. 项目概览

| 项目 | 说明 |
| ------ | ------ |
| 项目名称 | TravelVista |
| 项目类型 | 静态旅游目的地浏览网站 |
| 目标用户 | 旅游爱好者、行程规划者 |
| 主语言 | 中文（附英文版本） |
| 设计理念 | 简洁清晰、蓝绿色调、响应式布局 |

### 项目目标

为用户提供精美的旅游目的地浏览体验，展示全球热门旅行目的地。用户可通过搜索和筛选快速找到感兴趣的目的地，查看详细信息，获取实用旅行参考。

---

## 2. 设计规范

### 配色方案

| 用途 | 色值 | 说明 |
| ------ | ------ | ------ |
| 主色-深蓝 | `#1E40AF` | 导航栏、标题、主按钮 |
| 主色-蓝 | `#3B82F6` | 链接、hover 效果 |
| 辅色-深绿 | `#059669` | 辅助按钮、标签 |
| 辅色-绿 | `#10B981` | 成功状态、强调元素 |
| 背景色 | `#F0F9FF` | 页面浅蓝背景 |
| 文字-主 | `#1E293B` | 正文文字 |
| 文字-次 | `#64748B` | 辅助说明文字 |

### 字体

- 系统默认 sans-serif（Tailwind 默认字体栈）

### 响应式断点

| 断点 | 宽度 | 说明 |
| ------ | ------ | ------ |
| Mobile | < 640px | 单列布局，汉堡菜单 |
| Tablet | 640px – 1024px | 双列卡片网格 |
| Desktop | > 1024px | 三列卡片网格，完整导航 |

### 图片来源

- **Unsplash** (<https://unsplash.com>)
- **Pexels** (<https://www.pexels.com>)
- 所有图片下载至本地 `assets/images/` 目录

## 3. 页面与功能说明

### 3.1 首页（index.html / index-en.html）

| 区域 | 功能描述 |
| ------ | --------- |
| 导航栏 | Logo + 首页/目的地/关于我们 + 语言切换（中/EN） + 移动端汉堡菜单 |
| Hero 区域 | 全屏背景大图 + 主标语 + 搜索框（关键词实时筛选目的地） |
| 热门目的地 | 卡片式网格，展示 8 个热门目的地（图片 + 名称 + 简介） |
| 分类导航 | 4 个旅行主题分类：海滩、山岳、城市、文化 |
| 用户评价 | 轮播组件，展示 3-4 条用户旅行评价 |
| 页脚 | 版权信息 + 快速链接 + 社交媒体图标 |

**搜索功能说明**：输入关键词后，实时过滤首页热门目的地卡片，匹配目的地名称和描述。

### 3.2 目的地列表页（destinations.html / destinations-en.html）

| 区域 | 功能描述 |
| ------ | --------- |
| 搜索栏 | 关键词输入框，支持实时过滤 |
| 筛选栏 | 地区下拉（亚洲/欧洲/北美/南美/非洲/大洋洲）+ 类型下拉（海滩/山岳/城市/文化） |
| 卡片网格 | 响应式网格：mobile 1列 / tablet 2列 / desktop 3列 |
| 卡片内容 | 封面图、目的地名、国家、简短描述、评分星级 |
| 无结果提示 | 筛选无匹配时显示友好提示信息 |

### 3.3 目的地详情页（destination-detail.html / destination-detail-en.html）

| 区域 | 功能描述 |
| ------ | --------- |
| 图片画廊 | 大图展示区，展示 3-4 张目的地图片 |
| 基本信息 | 名称、位置、描述、最佳旅行季节、平均花费 |
| 景点推荐 | 3-4 个必游景点，含图片和简介 |
| 实用信息 | 交通方式、签证要求、当地货币、时区 |
| 相关推荐 | 3 个相关目的地卡片 |

### 3.4 关于我们页（about.html / about-en.html）

| 区域 | 功能描述 |
| ------ | --------- |
| 使命愿景 | 网站理念与目标说明 |
| 团队介绍 | 3-4 位团队成员照片 + 姓名 + 职位 |
| 联系方式 | 邮箱、社交媒体链接 |

## 4. 公共组件

### 导航栏（Nav）

- Logo（TravelVista）
- 导航链接：首页、目的地、关于我们
- 语言切换按钮：中 / EN
- 移动端：汉堡菜单图标，点击展开/收起导航

### 页脚（Footer）

- 版权信息（© 2026 TravelVista）
- 快速链接：首页、目的地、关于我们、隐私政策
- 社交媒体图标：微博、微信、Instagram

### 目的地卡片（Card）

- 封面图片（16:10 宽高比，`object-cover`）
- 目的地名称（加粗）
- 国家/地区标签
- 简要描述（2 行截断）
- 评分星级
- 圆角 `rounded-lg`、阴影 `shadow-md`、hover 放大效果

### 按钮样式

| 类型 | 样式 |
| ------ | ------ |
| 主按钮 | 蓝色背景 `bg-blue-700` + 白色文字 |
| 辅按钮 | 绿色背景 `bg-emerald-600` + 白色文字 |
| 幽灵按钮 | 透明背景 + 蓝色边框 + 蓝色文字 |

## 5. 技术栈

| 技术 | 说明 |
| ------ | ------ |
| HTML5 | 语义化标签（header, nav, main, section, footer） |
| Tailwind CSS | 通过 CDN 引入，utility-first 样式框架 |
| 原生 JavaScript | 汉堡菜单、轮播、搜索筛选、语言切换 |
| 图片资源 | 从 Unsplash/Pexels 下载到本地 |

**不使用**：前端框架（React/Vue）、构建工具（Webpack/Vite）、后端服务

## 7. 响应式设计要求

- **Mobile-first** 开发策略
- 导航栏在移动端折叠为汉堡菜单，点击展开
- 卡片网格使用 Tailwind 响应式类：`grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`
- 图片使用 `object-cover` + 固定宽高比，保持视觉一致性
- Hero 区域在移动端调整高度和文字大小
- 筛选栏在移动端堆叠为垂直布局

---

## 8. 多语言支持

- 中文为主语言，所有 `-en.html` 文件提供对应英文版本
- 导航栏右侧提供语言切换按钮
- 点击语言按钮跳转到对应语言版本的同一页面
- 中英文版本共享相同图片资源和 JS 逻辑
